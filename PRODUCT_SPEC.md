# Product specification

**Target user:** a person working through one question, decision, hypothesis, or unfinished idea.

**Golden path:** enter a prompt, run parallel Order and Curiosity lenses, read their distinct structured cards, then use Synthesis to choose a small next experiment. The user can copy any result, revise the prompt, clear the session, or retry a failed lens.

**Optional comparison:** after Synthesis, the user may request one concise Standard Response for structural comparison. It is an additional provider call, not an automatic action or controlled benchmark.

**Definition of done:** the page is responsive and keyboard-accessible; demo mode works without a key; live requests are server-side, structured-output validated, and provider-isolated; failures are clear; documentation and automated critical-logic tests are present.
