# Submission notes

## One-sentence pitch

Rebis Engine separates evidence-aware analysis from possibility-making, then turns their tension into a small testable next step.

## 100-word description

NeuroCuria: Rebis Engine is a dual-lens reasoning instrument built for unfinished questions. A single prompt is examined by Order, which identifies supported conditions, assumptions, risks, and verification needs, and Curiosity, which expands plausible reframes and possibilities. Synthesis then integrates both lenses into a recommended path, small experiments, and remaining uncertainty. The interface makes those modes visibly distinct without presenting them as separate personalities or authorities. It is an experimental Build Week prototype with a clearly labeled demo mode, structured validation, and accessible controls.

## 250-word description

Most AI interfaces merge analysis, imagination, and recommendation into one confident-looking answer. NeuroCuria: Rebis Engine makes that blend inspectable. A user brings one decision, problem, hypothesis, or unfinished idea. Order evaluates what is actually supported in the supplied material, highlights assumptions and risks, and names what needs verification. Curiosity stays connected to the same prompt while producing alternative interpretations, grounded and exploratory possibilities, and explainable cross-disciplinary connections. Synthesis uses both structured outputs to identify genuine agreement, preserve productive tension, qualify unsupported claims, and propose small reversible experiments.

The result is not a chatbot performance or a claim of separate trained minds. It is a practical reasoning interface based on deliberate cognitive separation. Its dark, technical visual system gives Order and Curiosity distinct forms and brings them together through a restrained convergence panel. The prototype runs in a no-key demo mode using disclosed sample responses, with an optional server-side OpenAI provider for live analysis. Synthesis does not run without both lens results. Rebis is an experimental thinking aid, not a source of medical, legal, financial, crisis, or other qualified professional advice.

## Key differentiators

1. Separate, inspectable evidence and possibility lenses.
2. Structured outputs that turn reasoning into cards and experiments.
3. A deployable demo path that is transparent about sample data.
